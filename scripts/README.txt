Parse contents of a xlsx file.

Use ./parse.sh <file> to get the averages of the specified xlsx file, tranposes rows, and saves in file "averages.csv"
Use ./spaces_to_underscores.sh to replace all spaces in filenames with underscores in the current directory

Scripts to call: 
./parse.sh <file>
./spaces_to_underscores
py awkcols <column_name_1> <column_name_2> ...
